# Behavior

## Hover Effects
- Table rows: background highlight on hover (#F1F5F9)
- Metric cards: subtle shadow elevation on hover

## Status Dot
- Pulsing animation on "active" status agents
- Colors: active=#10B981, idle=#64748B, error=#EF4444

## Sorting
- Table supports sortColumn and sortDirection props
- Visual indicator on active sort column header

## Chart Interactivity
- Hover tooltip showing exact value (optional, not required for MVP)

## Responsive
- Breakpoints: 1440px (full), 1024px (compressed), 768px (stacked)
- Tables: horizontal scroll on small screens
